# Renace Hats – Sitio Web de Catálogo https://renacehats.com/

Sitio web de catálogo corporativo desarrollado para Renace Hats, una empresa de sombreros que busca establecer presencia en internet y mostrar de forma clara sus productos y la identidad de la marca.

El proyecto consiste en una aplicación front-end enfocada en la presentación visual del catálogo, información relevante de la empresa y canales de contacto, con un diseño responsivo y una estructura sencilla orientada a la experiencia del usuario.

Características principales:

Sitio web de catálogo

Diseño responsivo para dispositivos móviles y escritorio

Presentación de productos e información corporativa

Estructura clara y navegación sencilla

Optimización básica para visibilidad en buscadores (SEO)

Objetivo del proyecto:
Ofrecer una vitrina digital funcional y profesional que permita a la empresa mostrar su oferta de productos y fortalecer su presencia online.
